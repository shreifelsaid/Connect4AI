Text-terminal based game to play connect4. Uses h-minimax with alpha-beta pruning.

To run:

$java Play


The game performs best on intermidate, giving almost instant feedback, it takes 3-4 seconds on expert.
It does this by implementing a h-minimax with alpha beta pruning.

Connect 3 performs perfectly [cannot lose, gaurnteed a draw as connect3 is a solved game] and quickly.
